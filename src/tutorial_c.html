ESTRUTURAS CONDICIONAIS

IF

É uma estrutura de decisão que verifica se uma ou mais condições são verdadeiras. Se for verdadeira, executa-se o bloco de código. Se não for, não executa-se o bloco de código.
Exemplo: 

                if (condição) {
                    // bloco de código
                }

IF-ELSE

O ELSE é usado nas situações onde existe a possibilidade daquela condição não ser atendida. Se a condição for verdadeira, o bloco de código do IF é executado. Se não for, o bloco de código do ELSE é executado.
Exemplo: 

                if (condição) {
                    // bloco de código
                } else {
                    // bloco de código
                }

SWITCH

O SWITCH é usado para fazer a seleção de uma ou mais condições.
Exemplo: 

                switch (condição) {
                    case 1:
                        // bloco de código
                        break;
                    case 2:
                        // bloco de código
                        break;
                    default:
                        // bloco de código
                        break;
                }
